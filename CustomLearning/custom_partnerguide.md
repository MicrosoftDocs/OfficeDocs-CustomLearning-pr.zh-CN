# <a name="microsoft-365-learning-pathways-early-adopter-program"></a>Microsoft 365 学习路径早期采用者计划

如果你是有兴趣与同行和学习路径工程团队联系的合作伙伴或 MVP，我们将邀请你加入早期采用者计划。 此社区侧重于协作环境，让你能够提出有关以下指南的问题以及向同行学习。 有关计划需要哪些内容，更具体包括：  
- 活动合作伙伴和 MVP 协作参与论坛 
- 每月载入和认知主题 
- 路线图见解 
- 提前访问合作伙伴载入指南和工具

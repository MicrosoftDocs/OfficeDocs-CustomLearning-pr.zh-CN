# <a name="microsoft-365-learning-pathways-early-adopter-program"></a>Microsoft 365 学习路径早期采用者计划

如果你有兴趣联系你的同行和学习路径工程团队的合作伙伴或 MVP，则我们会邀请你加入我们的早期采用的先期采用计划。 此社区重点关注协作环境，使您能够在以下指导中提出问题，以及从同行了解。 有关该程序所需内容的更多详细信息包括：  
- 主动合作伙伴和 MVP 协作合作论坛 
- 每月载入和知晓主题 
- 路线图见解 
- 早期访问合作伙伴加入指南和工具

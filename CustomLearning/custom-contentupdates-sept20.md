# <a name="learning-pathways-september-2020-content-updates"></a>学习2020年9月内容更新的途径

- 确保家庭计算机安全
- 使用 Windows 安全帮助保护你的家庭在线–你是否已在此处意外使用此功能？

### <a name="security-at-work-playlist"></a>"工作中的安全性" 播放列表

- 防止安全设置中的更改具有防篡改保护
